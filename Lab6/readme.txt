-> сделано под ОС Linux
-> make clean - убирает все скомпиленные файлы
-> в качестве дополнения реализована ф-я:
void* realloc(void* ,size_t );
-> в качестве демонстрации работы пользователю предлагается динамически заполнить массив чисел. 
Пока не будет введён 0, для массива будет динамически перевыделятся память и добавлятся новое введённое число.
После ввода 0 на экран выводится весь полученный массив, затем эта память освобождается и программа завершается.
